1. 應達目標為規劃日期計算至前二日的應達目標。為因應長期投資且報酬率符合預期的假設下，資產規模將出現複利的拋物線特性，應達成之獲利金額(目標金額-已投入金額)採取總投資天期前後期間45：55的比例設定，再依前後段的投資期間平均分攤，故目標理財之投資前半期間的應達目標為：已投資金額+45%*(目標金額-已投入金額)*(規劃日期前二日-投組成立日期)/(總投資天期*50%)，後半期間的應達目標再將應達成獲利的剩餘金額按時間比例分攤計算。若僅做定期定額投資，定期定額方式之應達目標公式為將上式的”已投入金額”置換為”定期定額到期時總預定投入金額”。
  2. 目前進度狀態說明：(1)符合進度：達成率 ≥ 100% (2)微幅落後：90%<=達成率<100% (3)落後：達成率<90%。
  3. 達成率=市值(含息)/應達目標；含息報酬率=(市值(含息)-投入金額)/投入金額；市值同時包含基金參考市值與連結全委帳戶之投資型保險保單帳戶價值，且此數值僅供參考。
  4. 配息之計算僅包含基金配息金額，不包含連結全委帳戶之投資型保險配息金額，如欲進一步瞭解請洽本行理財專員或電洽富邦人壽客戶服務專線0809-000-550。
